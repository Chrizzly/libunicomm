need_replyWhether reply needed for this message. This affects the timeouts processing. If reply is not needed the timeout for the send message is just ignored even if the timeouts enabled and timeout value is specified for the message.
toutMessage reply waiting timeout.
answersAllowed answers list. Empty list allows everything.
priorityDefault priority value for the message. If not explicitly set by the user's code this value is assigned. Only if unicomm::config::use_default_message_priority() is set.
