Detailed Description

template<typename DerivedT, typename SessionParamsT = void_session_params, typename BaseT = session_base>
class unicomm::extended_session< DerivedT, SessionParamsT, BaseT >

Session class which allows to map callable objects to events.

Template Parameters
DerivedTType that inherits extended_session.
SessionParamsTParameters type that will be passed to the session constructor. By default void_session_params is used.
BaseTBase type for the session. Default is session_base.
