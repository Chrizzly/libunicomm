Known issues
The debug-ssl configuration linkage failure under Windows
If boost naming layout is versioned the debug-ssl variant is not built in coulple with –use-pre-built-boost option. Use boost as source tree (by default) if you need this variant to be built.
Note: The boost system layout is not tested.
Recommendations
If you use static unicomm::server or unicomm::client object be sure to stop it before object starts destroying. Undefined behaviour like hanging up and memory access violations may occur due to undefined destroying order of static objects.
