Detailed Description

template<typename DerivedT, typename SessionParamsT = void_session_params, typename BaseT = session_base>
class unicomm::basic_session< DerivedT, SessionParamsT, BaseT >

Session interface to be implemented by user's code to receive event notifications.

This is base class for user's objects. It receives different notifications about the communication process. This is the place where the business logic should live.

Template Parameters
DerivedTType that inherits unicomm::basic_session.
SessionParamsTParameters type that will be passed to the session constructor. By default unicomm::void_session_params is used.
BaseTBase type for the session. Default is unicomm::session_base.
Note
Throwing an exception from any allowed handler interrupts current process iteration for the session object. An attempt to call left handlers will take place on next process iteration.
See Also
Sessions, unicomm::session_base, unicomm::extended_session.
