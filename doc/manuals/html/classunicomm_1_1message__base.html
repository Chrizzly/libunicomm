static const std::string& unicomm::message_base::message_format ( void  )
inlinestatic

Message format identifier implemented by this class.

Message format is just a string (identifier) which tells the engine to use predefined mechanisms or user implemented. Can have any value. This is optional feature and only have effect with unicomm::setup_message_format() and unicomm::load_from_complex_xml().

Returns
Message format name this class is regarded to.
See Also
setup_message_format(), load_from_complex_xml(), config::message_decoder(), config::message_encoder().
